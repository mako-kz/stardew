# Completed/Defunct Projects

## Mako's Spam Mail
([Nexus Link](https://www.nexusmods.com/stardewvalley/mods/22765) ♢ [Github](https://github.com/mako-kz/stardew/tree/main/SpamMail))  
A small mod that adds a number of mostly-jokey letters to the player's mailbox at random times. The aim was to give the player a few extra surprises throughout the game in a low-impact manner.  
The code is still functional as of SDV version 1.6, but is likely to break as time progresses as it was written in event syntax (not suggested, better for older versions like 1.5) instead of trigger actions (recommended for 1.6 and onward). The code remains here for legacy purposes.  
Requires [Content Patcher](https://www.nexusmods.com/stardewvalley/mods/1915).
