# Wayward Wings

An in-progress expansion for Stardew Valley themed around dragons. Currently the mod only adds one character, Argent, and not much to do with him. This mod is *extremely* incomplete and is not intended for download yet (but I also can't stop you from doing such).  

This mod has a Trello board, accessible [here](https://trello.com/b/asPZRmBJ/wayward-wings). It is currently heavily out of date as I am coming back to the project after a couple of years and am evaluating my progress.

## Configs
# Gameplay-altering configs
**TimeBetweenLetters:** Configures how much time needs to pass between casual letters. Casual letters are a system introduced by this mod that is not fully implemented yet. Default is seven days.

# Narrative-altering configs
**NonHumanFarmer:** Alters some dialogue throughout the mod for roleplay or immersion purposes. Some things may play out different depending on this setting. If you consider your character to be either nonhuman or immortal, enable this setting for the best experience.  
Default is disabled.

**Pronoun Selection Configs:** A set of configs for selecting the character's pronouns. Defaults to they/them pronouns.

## Dependencies
This mod was originally started in 2024 as a potential candidate for East Scarp, a collaborative expansion. While I am still deciding if I will submit the project for consideration, it is currently a dependency as I continue to work on the mod in general.  
As of right now, this mod requires the following mods to work:
- [Content Patcher](https://www.nexusmods.com/stardewvalley/mods/1915)
- [SpaceCore](https://www.nexusmods.com/stardewvalley/mods/1348)
- [East Scarp](https://www.nexusmods.com/stardewvalley/mods/5787)
